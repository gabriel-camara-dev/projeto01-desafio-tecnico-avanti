# Projeto 01 - Desafio Técnico Avanti

Objetivo: Implementação do layout proposto no teste técnico da Avanti (Innovation Class).

## Tecnologias

- HTML
- JavaScript
- Tailwind CSS
- Swiper.js

## O que foi implementado

- Estrutura HTML seguindo o layout do Figma
- Layout responsivo (desktop e mobile)
- Busca no cabeçalho com exibição da mensagem:
    Você buscou por: 'assunto buscado'
- Carrosséis com navegação e paginação usando Swiper

## Como rodar

Opções:

1. Live Server no VS Code
2. npx serve

Depois, abra a URL local gerada (ex.: http://localhost:5500).

## Observação

O projeto foi componentizado em arquivos HTML parciais para manter a organização e evitar um `index.html` muito grande.
